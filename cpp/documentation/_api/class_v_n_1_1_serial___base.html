virtual Error open(const PortName &portName, const uint32_t baudRate) noexcept = 0¶

Opens communication to a serial port. No bytes are to be transmitted.

Parameters:

  • portName – Name of the port to open. On Windows, an example is “COM13”

  • baudRate – The baud rate at which to communicate. On Windows, an example is 115200.

virtual void close() noexcept = 0¶

Closes communication to a serial port, leaving it available to be reopened. No data can be sent or read.
