class EigenDecomposition¶

Class for performing Eigen Decomposition of a matrix. This class provides an implementation for computing the eigenvalues and eigenvectors of a square matrix.

Public Functions

inline EigenDecomposition()¶

Default constructor for EigenDecomposition. The default constructor sets the maximum number of iterations to 50.

inline EigenDecomposition(int maxIter)¶

Constructor for EigenDecomposition with custom number of maximum iterations.

Parameters:

maxIter – The maximum number of iterations for the decomposition.

Computes the eigenvalues and eigenvectors of a square matrix.

Template Parameters:

  • m – The size of the matrix (m x m) and vectors (m x 1).

  • T – The type of elements in the matrix and vectors.

Parameters:

  • mat – The square matrix.

  • eigenValuesR – [out] The real parts of the eigenvalues.

  • eigenValuesI – [out] The imaginary parts of the eigenvalues.

  • eigenVectors – [out] The matrix containing the eigenvectors.

Returns:

Indicates if an error occurred. True if an error occurred, false if successful.
