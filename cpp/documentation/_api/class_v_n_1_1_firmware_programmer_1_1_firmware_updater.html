class FirmwareUpdater¶

Class responsible for updating firmware on a VectorNav unit.

The FirmwareUpdater class provides functionality to update the firmware on a VectorNav unit. It handles all processor switching and supports updating firmware from either a specified VNXML or a list of VNX files.
